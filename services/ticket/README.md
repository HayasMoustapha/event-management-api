# Ticket Service

Port: 4003

## Description
Manages tickets and stock.

## Endpoints
- `GET /tickets/event/:eventId`
- `GET /tickets/:id`
- `GET /tickets`
- `POST /tickets/event/:eventId`
- `GET /tickets/:id/`
- `PUT /ticket/:id/`
- `DELETE /ticket/:id/`

## Environment
- MONGODB_URI

## Run
```bash
npm install
npm start
```
